# 5.6 语音合成（TTS）

> **核心问题**：2026 TTS 在延迟（Cartesia ~40ms）、表现力（ElevenLabs）、共情（Hume）、企业落地（Deepgram）四个方向各有强者，怎么选？什么时候自托管？
> **目标**：能为产品（IVR / 语音 agent / 有声内容 / 多角色配音）在 TTFA、自然度、多语种、克隆能力、价格之间选对模型。

**展开重点**：
- **2026 TTS 四大方向**：
  - **极致低延迟**：**Cartesia Sonic 4**（~40ms TTFA，对话首选）；Deepgram Aura-2（~90ms，企业首选）
  - **极致表现力**：**ElevenLabs v3**（4000+ 音色 / 70 语言）；语调、情感、停顿丰富
  - **情绪共情**：**Hume Octave 2**（emotional fidelity 标杆，适合心理 / 教育 / 陪伴）
  - **企业 / 私有部署**：**Deepgram Aura-2**（含 on-prem）
- **关键能力分项**：
  - **TTFA（time-to-first-audio）**：voice agent 必关注；Cartesia ≤ 50ms 是当前 SOTA
  - **音色克隆**：ElevenLabs 强、可商用；Hume / Cartesia 跟进
  - **多语种 / 多口音**：ElevenLabs 70 语种领先；商业 IVR 重点
  - **SSML / 标记控制**：停顿、重音、笑声；表现力场景必备
  - **流式输出**：所有现代 TTS 都需支持，否则不适合实时
  - **生成长度限制**：长篇有声内容看是否支持分段拼接
- **选型矩阵**：
  - Voice agent 对话 → Cartesia Sonic 4
  - 多角色 / 戏剧化朗读 → ElevenLabs v3
  - 心理 / 教育 / 陪伴场景 → Hume Octave 2
  - 企业内 / 通话 / 合规严 → Deepgram Aura-2（含 on-prem）
  - 开源自托管 → XTTS / Bark / OpenVoice / F5-TTS（成本敏感、批量场景）
- **统一平台趋势**：Deepgram（Aura-2 + Nova-3）、Cartesia（Sonic + Ink-Whisper）、ElevenLabs（TTS + Scribe v2）、Azure Voice Live —— **STT + TTS 一体化**降低延迟与集成成本
- **价格 / 计费**：按字符或秒计费；高表现力档显著贵；商用克隆音色有授权费
