# 个人技术博客（高信噪比作者）

除组织外，几位作者的个人博客长期产出**高信噪比、成体系**的深度长文，很多是行业公认的必读。

---

## 精选来源

- 🔴 [Lilian Weng](https://lilianweng.github.io/) — 前 OpenAI。以"综述级"深度长文著称（Agent、Prompt、幻觉、扩散模型等），一篇顶几十篇二手教程。→ **全部章节**
- 🔴 [Chip Huyen](https://huyenchip.com/blog/) — 《Designing Machine Learning Systems》《AI Engineering》作者。系统化讲 AI 工程、LLMOps、生产落地。→ **第 6、7 章**
- 🟡 [Eugene Yan](https://eugeneyan.com/) — Amazon 应用科学家。RAG、评估、LLM 应用模式的实战长文，落地味浓。→ **第 2、6 章**
- 🟡 [Simon Willison](https://simonwillison.net/) — 追更最勤的 LLM 实践观察者，新工具/新模型/prompt 注入等一手评测与踩坑。→ **第 1、8 章**
- 🟢 [Jason Liu](https://jxnl.co/writing/) — RAG 系统化改进、结构化输出（Instructor 作者）实战。→ **第 2 章**

---

## 备注

个人博客更新节奏和立场各异，但这几位的**方法论密度**通常高于机构博客。读 Lilian Weng / Chip Huyen 适合"建立体系"，读 Simon Willison 适合"跟上最新动态"。
