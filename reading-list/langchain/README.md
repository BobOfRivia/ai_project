# LangChain / LangGraph

> 入口：博客 <https://www.langchain.com/blog> · 文档 <https://docs.langchain.com>

主流 LLM 应用**编排框架**的官方博客。LangChain / LangGraph 均于 **2025 年 10 月发布 v1.0**（稳定 API、面向生产可靠性）。内容偏"用框架把 Agent / RAG / 工作流搭起来"的工程实践，也发布行业趋势报告。

**覆盖主题**：LangGraph 状态图编排、Agentic RAG、Human-in-the-loop、持久化 checkpoint、可观测（LangSmith）、生产可靠性。

---

## 精选必读

- 🔴 [State of Agent Engineering](https://www.langchain.com/state-of-agent-engineering) — 基于 1340 份问卷（2025.11–12）的年度报告，看清行业在怎么建 agent。→ **第 3 章**
- 🟡 [Beyond RAG: Implementing Agent Search with LangGraph](https://www.langchain.com/blog/beyond-rag-implementing-agent-search-with-langgraph-for-smarter-knowledge-retrieval) — 并行化、状态管理、可扩展企业 agent 的实践。→ **第 2、3 章**
- 🟡 [Build a Custom RAG Agent with LangGraph](https://docs.langchain.com/oss/python/langgraph/agentic-rag) — 官方教程：让 agent 自己决定何时调用检索工具。→ **第 2、4 章**
- 🟢 [LangChain Blog](https://www.langchain.com/blog) — 持续追更框架新特性、集成、案例。
