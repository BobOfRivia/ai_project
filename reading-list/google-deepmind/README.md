# Google / DeepMind

> - **Google Research 博客**：<https://research.google/blog/>
> - **Google DeepMind 博客**：<https://deepmind.google/discover/blog/>
> - **Google AI for Developers（Gemini）**：<https://ai.google.dev>

Google 体系偏**研究前沿 + 大模型/多模态工程**。DeepMind 侧重前沿研究（含 Gemini、多模态、Agent），Google Research 侧重可落地方法，AI for Developers 是 Gemini API 的实操文档。

**覆盖主题**：Gemini / 多模态、长上下文、检索与嵌入研究、Agent、负责任 AI。
