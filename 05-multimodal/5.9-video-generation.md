# 5.9 视频生成 API

> **核心问题**：**Sora 2 已宣布弃用**（2026-04-26 宣布、2026-09-24 关闭），生产管线该迁到哪里？Veo 3.1 / Kling 3.0 Omni / Seedance 2.0 / Runway Gen-4.5 / Pika / HappyHorse 各自的位置是什么？
> **目标**：能为业务（广告短片 / 社媒短视频 / 产品演示 / 角色对白 / 多镜头叙事）选对模型、聚合器与时长 / 价格档位。

**展开重点**：
- **2026 视频生成格局**：
  - **Sora 2 / OpenAI**：2026-04-26 宣布弃用，Videos API 2026-09-24 关闭 —— 任何依赖 Sora 的生产管线**必须迁移**
  - **Google Veo 3.1**：整体质量第一档；唯一原生 4K；60s clip；fast 模式起 $0.15/sec
  - **Kling 3.0 Omni**（快手）：运动质量领先；**多镜头 + 同步音频 + 5 语种对白**；最长 2 分钟；~$0.10/sec
  - **Seedance 2.0**（字节 ByteDance，2026-02-12）：叙事多镜头 + 音视频一体；Artificial Analysis 榜首
  - **HappyHorse-1.0**（阿里 ATH，2026-04）：与 Seedance 占据 AA 榜前两位
  - **Runway Gen-4.5**：image-to-video 标杆；reference image + 相机控制；API 生态最成熟；16s 上限
  - **Pika**：社交内容向（Pikaffects / Pikaswaps / Pikadditions / Pikaformance lip-sync）
- **聚合器**：**fal.ai** 是事实入口，600+ 模型含 Kling 3.0、Seedance 1.5 Pro、Veo 3.1、Sora 2、Wan 2.6；$0.05–0.40/sec
- **选型矩阵**：
  - 极致质量 / 4K → Veo 3.1
  - 多镜头叙事 + 同步音频 → Kling 3.0 Omni / Seedance 2.0
  - image-to-video 精控 → Runway Gen-4.5
  - 社媒 / 病毒短视频 → Pika
  - 自由换模型 / A/B → fal.ai
- **关键能力分项**：
  - **时长**：Kling 2 分钟 > Veo 60s > Sora 25s > Runway 16s
  - **音频同步**：Kling Omni / Seedance / Veo 3.1 现已三家原生支持
  - **多镜头**：Kling / Seedance
  - **角色 / 风格一致**：reference image（Runway / Kling）
- **接入要点**：
  - 价格按"秒"计费，几秒到几十秒就是几美元；预算需提前估
  - 生成时长（wall clock）通常 1–10 分钟；走异步 + webhook，不要 block 请求
  - safety filter 与版权风险（角色、品牌、名人）
  - 输出格式（mp4 / webm / HLS）、分辨率与水印
- **2026 迁移建议**：任何用 Sora 2 的现存系统在 2026-09-24 之前完成切换；Veo / Kling / Seedance / Runway 是四个主候选
