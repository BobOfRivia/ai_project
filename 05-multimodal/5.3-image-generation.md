# 5.3 图像生成 API

> **核心问题**：2026 三层格局（Premium / Open-weight / Aggregator）里怎么选？GPT Image 2 vs Midjourney V8 vs Flux 2 vs Imagen 4 vs SD 3.5 各自的位置是什么？
> **目标**：能为业务（营销素材 / 头像 / 产品图 / 内部插图等）在质量、价格、商用许可、可控性之间选对模型与接入方式。

**展开重点**：
- **2026 三层格局**：
  - **Premium**：**GPT Image 2**（2026-04 上线，替代 DALL-E 3；引入"推理步"，文字渲染与复杂指令显著提升）、**Midjourney V8**（2026-03，引擎重写、5× 速度、原生 2K）、Imagen 4
  - **Open-weight**：**Flux 2**（Black Forest Labs，2026 现代开源主力）、Stable Diffusion 3.5、Ideogram 3（文字渲染强）、Recraft V3（设计强）
  - **Aggregator**：fal.ai、Replicate、Together、Fireworks、Stability API —— 同一套模型大幅降价，应用工程师的默认接入方式
- **价格三档（2026）**：
  - Premium：$0.03–0.20 / 图
  - Open-weight 原厂 API：$0.02–0.10 / 图
  - 聚合器托管的同款 open-weight：$0.008–0.04 / 图
- **按场景选择**：
  - 文字渲染（海报、UI 截图、宣传图含字）：GPT Image 2 / Ideogram 3
  - 艺术 / 设计感：Midjourney V8 / Recraft V3
  - 商用许可 + 灵活部署：Flux 2（Apache 2.0 的 schnell + Pro API 组合）
  - 极致便宜 / 量大：SD 3.5 走聚合器
  - 一键接入 + 多模型 A/B：fal.ai
- **接入维度**：
  - 商用许可（DALL-E / GPT Image 受 OpenAI ToS；Flux schnell Apache 2.0；SD CreativeML Open RAIL-M）
  - Safety filter 严格度（Premium 普遍更严，可能拒绝商业素材）
  - 输出分辨率与多张 batch
  - prompt 一致性（多次生成同一风格 / 角色）—— 这部分常常需要走 5.4 的"编辑 / 可控生成"
- **不放本节**：图像编辑、ControlNet、多步迭代 → 5.4
